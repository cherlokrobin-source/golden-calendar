Overview

The Golden Calendar (Digital Astrolabe Engine) is an advanced chronology engine designed to reconstruct and synchronize solar and lunar calendars over an unprecedented range of 50,000 years.

Inspired by the scientific legacy of the astrolabe, this project combines astronomy, chronology, and modern computation into a unified digital time engine capable of indexing millions of calendar records with deterministic precision.

---

Main Features

- 50,000-year solar chronology
- Linked lunar chronology
- Solar ↔ Lunar date conversion
- Day-by-day indexed archive
- Fast search by day number
- Fast search by solar date
- Fast search by lunar date
- Chronology API
- Express.js web server
- Structured archive system
- Batch generation architecture
- Scientific time synchronization
